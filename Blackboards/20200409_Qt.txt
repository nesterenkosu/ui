Добавление виджета на форму:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1). Подключить заголовочный файл в prog.h. Каждый виждет имеет отдельный заголовочный файл, имя которого совпадает с именем виджета.
QPushButton - кнопка
2). Объявить переменную объекта. Название виджета, его заголовочного файла и класса совпадают.
3). Создать экземпляр объекта.

Вывод кириллических строк - метод tr("строка")

Обработка событий:
~~~~~~~~~~~~~~~~~~
1). Объявить сигнатуру обработчика события в prog.h в секции slots
В терминах Qt обработчики событий называются СЛОТАМИ, а сами события - СИГНАЛАМИ.
2). Объявить обработчик в файле prog.cpp
3). Назначить обработчик события виджету.
connect(
	виджет,
	SIGNAL(обрабатываемое_событие),
	объект_содержащий_обработчик,
	SLOT(имя_обработчика)
)

Компоновка:
~~~~~~~~~~~
1). Компоновочные виджеты
QVBoxLayout - вертикальная компоновка
QHBoxLayout - горизонтальная компоновка
QGridLayout - компоновка в виде таблицы

->addWidget(виджет) - добавить виджет к компоновке
->addLayout(виджет) - добавить вложенный компоновочный виджет

Создание проекта в QtCreator
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Create Project > Qt Widget Project > Qt Gui Application
