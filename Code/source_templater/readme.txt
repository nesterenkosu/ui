Каталог содержит скрипты для быстрого создания заготовок программного кода на 
основе шаблонов, хранимых в каталоге FileTemplates.

ctpl – заготовка программы на языке C
gladetpl – заготовка программы на основе библиотеки Gtk+ с поддержкой
редактора графического интерфейса Glade
gtkidetpl – заготовка проекта Gtk+ - приложения с поддержкой Glade,
отлаживаемого в IDE QtCreator (создаёт ПАПКУ)
gtktpl – заготовка Gtk+ - приложения (без поддержки Glade)
qttpl – заготовка приложения на основе библиотеки Qt (создаёт ПАПКУ)
shtpl – заготовка скрипта интерпретатора tcsh

Формат вызова скриптов:
> скрипт имя_создаваемого_файла
Пример:
> ctpl myprogram.c
> gtkidetpl mygtkproject
